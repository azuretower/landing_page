Recipe Ingredient Scaler

This project allows users to store recipes and search through them based on several key points, the most interesting is searching by ingredients. You can enter a commna seperated list of ingredents and it will return the recipes that have those ingredients, sorted based on how many matches there were.

  • GitHub: recipes
  • Live Version: recipe.ryanmerrill.me
